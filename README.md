# Projeto Lojinha

Este projeto foi desenvolvido no âmbito acadêmico para a prática de programação. Ele utiliza tecnologias web como HTML, CSS, JavaScript e Bootstrap para a criação de uma interface interativa e responsiva.

## Tecnologias Utilizadas

- **HTML**: Estruturação do conteúdo da página.
- **CSS**: Estilo e design visual da página.
- **JavaScript**: Funcionalidades interativas.
- **Bootstrap**: Framework para construção de layouts responsivos e componentes prontos.

## Descrição do Projeto

O objetivo principal deste projeto é aplicar os conceitos aprendidos em aula para criar uma aplicação funcional e responsiva, utilizando boas práticas de desenvolvimento web. A ideia foi focar em tornar o projeto visualmente atraente e intuitivo para o usuário.
